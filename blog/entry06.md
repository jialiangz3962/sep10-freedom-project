### Challenges
Challenges I faced while making my website was adjusting the font sizes because bootstrap and Bulma overlapped with each other.

The code at first went like:
```css
h1 {
  font-family: "Oswald", sans-serif;
  font-optical-sizing: auto;
  font-weight: 700;
  font-size: 4rem;
}
```
But the input didn't work and the h1 stayed the same size. I solved the problem through the use of `!important`.

So I made changes to the code accordingly
```css
h1 {
  font-family: "Oswald", sans-serif !important;
  font-optical-sizing: auto !important;
  font-weight: 700 !important;
  font-size: 4rem !important;
}
```
